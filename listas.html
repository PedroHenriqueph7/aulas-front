Trabalhando com Listas

Listas são estruturas para representação de conteúdo na forma de relação de itens

Lista Não-Ordenada

Relação de itens em que a sequencia não importa.

Exemplos:

  • Teclado
  • mouse
  • monitor
  • Webcam

Lista Ordenada

Relação de itens em a sequencia importa.

Exemplos

  1. Andressa
  2. Bruno
  3. Carla
  4. Mônica
  5. Tiago

Listas aninhadas(listas dentro de Listas)

Podemos misturar listas de diferentes tipos e também colocar listas dentro de outras listas

Exemplos:

  1. Doces
    • Brigadeiro
    • Torta de Limão
    • Paçoca
  2. Salgados
    • Esfiha
    • Pizza
    • Coxinha

Lista de Definição

Lista usada para estrutura de itens baseados em termo e descrição. Útil para conteudos dos tipos: glossários, FAQs, ou apresentação de dados com alguns conceito etc.

Exemplo:

Processador
É a unidade de processamento computacional...

Peça essencial em qualquer equipamento digital.
Memória RAM
Espaço de armazenamento volátil...
